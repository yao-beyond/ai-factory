## 🍱 AI 實際做出來的（從產出的 `index.html` 驗證）

標題：**`餐飲 POS 原型`**，單一檔案約 **48 KB**，涵蓋我描述的四大功能：

- **點餐與出單**：桌號點餐、購物車、送出後依 **內場（廚房）／外場** 分區顯示待出單；含 **QR Code 掃碼** 自助點餐入口示意。
- **外送 / O2O 整合**：把 **Uber Eats、foodpanda、LINE** 訂單來源整合成同一張清單，示意「避免漏單」。
- **會員與行銷**：消費紀錄、**累積點數**、**分眾** 標籤。
- **庫存與報表**：即時 **銷售** 統計與 **食材庫存** 示意。
- 純前端、資料存 **localStorage**、手機/平板皆可用、介面正體中文。

（關鍵字實測出現次數：QR×14、庫存×12、會員×8、購物車×6、foodpanda×6、內/外場×各4、報表×4、出單×4、點數×3、銷售×3、桌號×3、Uber Eats×2、分眾×2…）

## ✅ 交付驗證（live）

- 👀 線上預覽 `GET /gateway/preview/{id}/` → **HTTP 200**（瀏覽器直接打開即用）
- ⬇️ 下載 `GET /gateway/result/{id}` → **HTTP 200**，附件 `ai-factory-<id>.zip`
- 📦 zip **乾淨**：只含專案本身（不含 `.git`、`.omc` 工具狀態、流程中介檔）

## 🧭 我（使用者）只做了三件事

1. 在首頁選「✨ 做一個全新的東西」，填標題與描述、選開發強度。
2. 在「📝 開工前確認」關卡看一眼白話計畫，按 **確認開工**。
3. 完成後按 **👀 線上預覽** / **⬇️ 下載你的專案**。

全程沒碰一行程式碼、沒設定任何 git 或金鑰。
